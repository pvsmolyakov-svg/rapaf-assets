# rapaf-assets

Static media for the RAPAF / ФВСАР site ([new.rapaf.ru](https://new.rapaf.ru)), served via the
free [jsDelivr](https://www.jsdelivr.com/) GitHub CDN so the application repo stays lean.

**CDN URL pattern:**

```
https://cdn.jsdelivr.net/gh/pvsmolyakov-svg/rapaf-assets@main/<path>
```

## Status

Legacy WordPress galleries (Envira) were migrated here on 2026-06-21, then dropped at the
owner's request (content no longer relevant). Reserved for upcoming asset migration: the
~213 page-hero and article-body images currently hotlinked from the old rapaf.ru host.
